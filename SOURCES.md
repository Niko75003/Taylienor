# Sources et crédits

## Catalogue musical
- Site officiel Taylor Swift : discographie et univers officiels.
- Apple Music / iTunes Search API : pochettes, métadonnées, tracklists et extraits de préécoute.
- MusicBrainz : contrôle des éditions et des tracklists.

Le site privilégie automatiquement une Taylor’s Version lorsqu’elle existe. Les tracklists complètes sont chargées depuis iTunes et mises en cache dans le navigateur. Une sélection locale reste disponible en solution de secours.
